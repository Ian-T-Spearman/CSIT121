What can you use SPARQL for?

SPARQL is mainly used for complex querying across RDF datasets, which includes filtering results, aggregating data, and combining different datasets. Although this is the main function of SPARQL, it can also be used update RDF databases and change the data that is stored, which is useful for more dynamic datasets. While also being very useful for machine learning, SPARQL is useful for linking data across datasets, which can allow for queries from different places on the web at the same time.

Main types of Queries:

There are many different types of queries available to use in SPARQL, however the most commonly used are:

  1. SELECT: Select matches found in a data graph.
  2. ASK: Ask if there is a match found in the dataset from the query.
  3. CONSTRUCT: Create new data from the existing data.
  4. DESCRIBE:Creates an RDF graph that 'Describes' the data.

SPARQL Query Example

SELECT ?subject ?predicate ?object
WHERE { ?subject ?predicate ?object . }

This is the template for a basic query, which when filled out would look something like:
SELECT ?personName ?cityName
WHERE { ?person rdf:type ex:Person . ?person ex:hasName ?personName . ?city ex:hasName ?cityName. }

This query searches the data and finds matches based off the links between 'Person' and 'City'.
